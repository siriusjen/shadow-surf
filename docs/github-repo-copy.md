# GitHub Repository Copy

这份文案用于 GitHub 仓库页面展示，目标受众是 IntelliJ / IDEA 用户，表达重点是：减少白底网页与暗色 IDE 的割裂感，以及在 IDE 内完成轻量网页访问。

## About Description

### 推荐版

Lightweight IntelliJ IDEA browser tool window that reduces bright-page contrast in dark themes and keeps lightweight browsing inside the IDE.

### 更短版

A lightweight IntelliJ IDEA browser tool window with dark-page enhancement.

## Short Intro

ShadowSurf is a lightweight in-IDE browser for IntelliJ IDEA. It helps you open pages without leaving the IDE and softens the visual clash between bright web pages and dark editor themes.
